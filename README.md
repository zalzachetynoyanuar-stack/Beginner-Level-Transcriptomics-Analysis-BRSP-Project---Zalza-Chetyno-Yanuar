## Topic Capstone Project: RSV infection transcriptional profiles vs. healthy controls

### Tools: GEO, R (R/Rstudio), GO (g:profiler), KEGG 
This project contains **beginner** transcriptomic analysis and analysis of the data obtained

## Steps for R Analysis using data from GEO ncbi
1. Check the journal and GEO database that we will explore (here we use GEO80179)
2. Check the platform of the selected GEO data
3. Ensure that R, R Studio, and Bioconductor are installed
4. Prepare the libraries and modules to be used (ensure the annotation database is compatible with the data platform to be analyzed)
5. Analyze using the required coding
6. Use visualization to make the data easier to analyze
